transform adjcent list¶

The adjcent list is created by collection of unordered movie lists that each UserID has rated. They represent a finite movie-interaction graph.

Each list describes the set of neighbors (movies that share common features in some degrees) of a vertex (movie) in the graph (network) based on users' behaviors.

adj_l = []
for col in ratings_unstack.columns:
    sub = ratings_unstack[ratings_unstack[col] == 1]
    sub = sub.drop(col,axis = 1)
    neibours = sub.columns[sub.sum(axis = 0) != 0]
    neibours_str = ' '.join(neibours.map(str))
    adj_row = str(col)+' '+neibours_str
    adj_l.append(adj_row)
